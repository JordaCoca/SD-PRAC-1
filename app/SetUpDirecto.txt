1️ - Levantar Redis
docker run -d --name myredis -p 6379:6379 redis
2️ - Load Balancer (entry point)
uvicorn app.load_balancer:app --host 127.0.0.1 --port 8080
3️ - Autoscaler (crea workers dinámicos)
curl -X POST http://127.0.0.1:8080/reset  ---> Esto para resetear el sistema y metricas
cd app
python autoscaler.py
4️ - Ejecutar benchmark
cd testing
python benchmark_runner.py
5 - Monitorizar workers
Con monitor.py podemos monitorizar los distintos workers
python monitor.py


Endpoint único:
http://127.0.0.1:8080
Flujo:
Benchmark → LB → Workers → Redis
Escalado:
Más carga → más workers
Menos carga → menos workers
